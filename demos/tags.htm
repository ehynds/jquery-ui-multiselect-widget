Tags Plugin

Tags are available by including the jquery.multiselect.tags.js plugin and the jquery.multiselect.tags.css CSS file. Initialize filtering on any of your multiselects by calling multiselecttags() on the widget.

The tags plugin displays a list of the selected elements in tag form below the multiselect :

$(function(){
   $("select").multiselect().multiselecttags(); 
});

Basic

Methods

Syntax: $("select").multiselecttags("method_name");

  • updateTags

    Call if you update the checked/selected state of the multiselect without trigering the "change" event on the select element.

  • destroy

    Destroys the widget.

  • widget

    Returns the wrapper ul with the tags inside. This is a quick and easy way to access the HTML created by the plugin.
