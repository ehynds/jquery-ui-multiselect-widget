OptGroup Collapsible and Selectable Demos

Multiselects are created with the following.

$(function () {
	$("select#one").multiselect({ optGroupCollapsible: true });

	$("select#two").multiselect({ optGroupSelectable: true });

	$("select#three").multiselect({ optGroupCollapsible: true, optGroupSelectable: true });
});
